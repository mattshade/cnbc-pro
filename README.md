# CNBC PRO Subscription Experience Case Study

A high-fidelity, interactive case study built to detail the redesign of the premium subscription journey for serious investors at CNBC PRO. This application is an immersive portfolio piece that showcases the end-to-end design process, from problem definition to final solution implementation, through a premium, responsive web interface.

## 🚀 Features

- **Dynamic Interactive Modals**: Cinematic, layout-animated modals that expand to showcase high-resolution UI prototypes, personas, and design systems using `framer-motion`.
- **Custom Pan & Zoom Engine**: A bespoke image viewer allowing users to deeply explore complex user journey maps and competitive analysis spreadsheets with intuitive click-to-zoom and pan functionality.
- **Rich Text OCR Viewer**: An editorial-style document viewer that elegantly renders digitized research artifacts (like Moderator Testing Scripts) directly within the DOM.
- **Premium Aesthetics**: Built with a sleek dark mode, vibrant green brand accents, subtle glassmorphism, and hardware-accelerated animations.
- **Dynamic Brand Favicon**: A completely custom SVG favicon that smoothly shifts hues over time to match the core engineering and design branding.

## 🛠️ Tech Stack

- **React 18**: Core component rendering.
- **Vite**: Blazing fast build tooling and dev server.
- **TypeScript**: End-to-end type safety.
- **Tailwind CSS**: Utility-first styling for rapid, responsive UI development.
- **Framer Motion**: Complex, layout-aware animations and physics-based interactions.
- **Lucide React**: Crisp, customizable SVG iconography.

## 📦 Local Development

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the dev server**:
   ```bash
   npm run dev
   ```
   Navigate to the local host port provided to view the case study locally.

3. **Build for production**:
   ```bash
   npm run build
   ```
   The optimized production bundle will be generated in the `dist` directory.

## 🧠 Design Philosophy

This project was built to transcend the standard "static PDF" or "long-scrolling image" case study format. By bringing the design artifacts to life through code, the portfolio piece itself becomes a demonstration of the engineering and design capabilities discussed within the study. Every interaction is highly intentional, aiming to reduce cognitive load while delivering a premium feel.
